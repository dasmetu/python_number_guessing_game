# Python Number Guessing Game

My first Python project.

This is a simple command-line number guessing game written in Python while learning programming fundamentals.

## Features

- Difficulty selection
- Integer input validation
- Higher / Lower hints
- Guess counter
- Replayability

## Skills practiced

- Functions
- While loops
- If / Else statements
- Exception handling (`try` / `except`)
- User input validation
- Random number generation

## How to run

```bash
python pythontestnotebook.py
```
